# Simple-School-Login-Enrollment-System-
Features : Login System, Forgot Password Function, Enrollment System with add remove Subject

How it works:

Create Acc system with global var contatining entered username and password 

Login System Checks if enterd string in text box matches with global var with the pre entered string in create acc 

Forgot Password rewrites the pre existing password with new string in textbox

Enrollment System stores all data and puts into a messagebox at the end 

Course Subject System if Selected course is picked shows a specific array to match the course
